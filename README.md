# Krish Kumar - Portfolio

A modern, sleek, and highly interactive personal portfolio showcasing my engineering projects, work experience, and blog.

## Tech Stack
- **Frontend:** React, Tailwind CSS, Vite, Redux Toolkit
- **Backend:** Node.js, Express, MongoDB
- **Design:** Custom neomorphic UI with a dynamic cursor, fully responsive light/dark themes.

## Features
- **Dynamic Custom Cursor:** Smooth following x-ray/lens effect on text and social icons.
- **Interactive Project & Experience Cards:** Detailed views with GitHub and live links.
- **Blog Section:** Integrated liking system.
- **Dark/Light Mode:** Seamless, animated view transitions between themes.

## Running Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/krish02git/Portfolio.git
   ```

2. **Frontend Setup:**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

3. **Backend Setup:**
   ```bash
   cd backend
   npm install
   npm run dev
   ```
